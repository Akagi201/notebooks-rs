# notebooks-rs

Data analysis with Rust Notebooks

## Install Evcxr Jupyter Kernel(Rust repl kernel)

```sh
cargo install --locked evcxr_jupyter
evcxr_jupyter --install
```

## Run jupyer-lab

```sh
jupyter lab
```
